## Image Generation Settings

For **FAL.AI** generation, use these parameters:
```json
{
  "model": "fal-ai/flux-pro",
  "image_size": "landscape_16_9",  // or "square" for buttons
  "num_inference_steps": 50,
  "guidance_scale": 7.5,
  "seed": -1,
  "prompt": "[PROMPT FROM ABOVE] + high quality, detailed, Hayao Miyazaki style, soft colors, hand-drawn appearance, warm atmosphere"
}
```

**Negative prompt** (add to all):
```
low quality, blurry, deformed, ugly, bad anatomy, bad proportions, scary, dark, violent, nsfw, text, watermark, signature
```
